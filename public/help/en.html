GPX Race Planner – Help

What the app does

GPX Race Planner lets you build a pacing plan for a race or training run. You can supply the route either by uploading a .gpx file or by entering segments manually with distance and elevation change. The result is a per-segment target pace list that can be exported as a workout directly into Garmin Connect.

Requirements

Google Chrome browser – importing the workout into Garmin Connect requires a Chrome extension that is only available for Chrome (and Chromium-based browsers such as Edge or Brave).

Installing the Chrome extension

To import the pacing plan into Garmin Connect, install the Share Your Garmin Connect extension:

↗ Open in Chrome Web Store

Once installed, the extension icon appears in Chrome. Click it to import the downloaded .json file directly into your Garmin Connect account.

Step-by-step – GPX file

1
Make sure the toggle is set to GPX file (default) and upload your .gpx route file by dragging it onto the drop zone or clicking to browse.
2
The app automatically splits the route into segments by gradient (uphill downhill flat). Adjust smoothing, slope threshold and min. segment length as needed.
3
Enter your target pace (min:ss / km) or target total time (h:mm:ss).
4
Choose an effort model (Linear / Power / Exponential / Minetti) and set uphill/downhill corrections. Optionally configure a split strategy (negative / even / positive).
5
Review the per-segment paces in the Pace plan table. Hovering a row highlights the corresponding segment on the map and elevation chart.
6
Click Download Garmin Workout (menu button or top-right). A .json file will be saved to your downloads folder.
7
Click the Share Your Garmin Connect extension icon in Chrome and import the downloaded .json into Garmin Connect.

Step-by-step – Manual entry

1
Switch the toggle to Manual.
2
For each segment enter the length (km) and elevation change (m or %). The segment type (uphill / downhill / flat) is detected automatically. Click + to add the segment.
3
Edit any segment with the pencil button – the form switches to edit mode. Confirm with the check button or Enter; cancel with Esc.
4
Continue the same way as the GPX flow (steps 3–7 above).

Effort models

  • Linear – pace scales directly with gradient. Simple and predictable.
  • Power – gentle slopes penalised less, steep slopes more. The exponent n controls the curvature.
  • Exponential – even stronger contrast between mild and steep gradients.
  • Minetti – biomechanical model (Minetti et al. 2002) with no user parameters. Most realistic but least configurable.

Split strategy

  • Negative split – start slower, finish faster.
  • Even split – constant effort throughout (default).
  • Positive split – start faster, slow down towards the end.

The split strength slider controls how pronounced the difference is between start and finish.
